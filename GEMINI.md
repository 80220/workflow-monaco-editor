# Description

Library containing customized components based on monaco editor.
Library is meant to be used as dependant library by other libraries.
It exports editor components.

## Project Instructions

- **Framework** we use React with Vite
- **Styling** put styles in .scss files
- **Tone** Be concise. Don't explain basic React concepts.
- **Linter** run 'npm run eslint' after code changge and fix any warnings/errors present
